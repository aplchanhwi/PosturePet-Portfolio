# PosturePet Portfolio

PosturePet은 macOS에서 일정 시간 동안 특정 앱을 사용하면 화면 위에 2D 캐릭터가 나타나 목/어깨 스트레칭을 알려주는 SwiftUI 기반 메뉴바 앱입니다.

SNS에서 본 화면 가림 휴식 유도 앱과 RunCat처럼 애착 가는 캐릭터 경험을 결합한 아이디어에서 출발했습니다. 사용자를 강제로 차단하기보다, 캐릭터 오버레이가 자연스럽게 끼어들어 짧은 휴식을 유도하는 경험을 목표로 만들었습니다.

이 저장소는 실제 서비스 앱의 포트폴리오 공개용 버전입니다.
실험용 스크립트, 로컬 StoreKit 테스트 설정, 사용자 캐릭터 런타임 데이터, Xcode 사용자 상태 파일은 제거했습니다.

## 주요 기능

- SwiftUI + AppKit 기반 macOS 메뉴바 앱 구조
- `NSWorkspace`를 활용한 활성 앱 감지와 사용 시간 누적
- 사용자가 감지할 앱을 선택하는 설정 화면
- 일정 시간 도달 시 화면 위에 표시되는 스트레칭 오버레이
- `나중에`, `완료` 액션과 미루기 시간 선택 UI
- 목 좌우 기울이기, 턱 당기기, 어깨 으쓱하기 스트레칭 루틴
- 메뉴바에서 스트레칭 루틴을 빠르게 선택하는 흐름
- 오버레이 창 크기와 위치 설정
- 기본 실루엣 캐릭터 애니메이션
- 캐릭터 슬롯 기반 커스터마이징 UI
- Vision/Core Image 기반 자동 배경 제거
- 이마, 턱, 목, 양쪽 어깨 기준점을 이용한 캐릭터 움직임 보정
- 한 장 이미지 기반 mesh deformation 미리보기
- StoreKit 기반 개별 슬롯 구매/월 구독 권한 모델
- 한국어/영어 앱 현지화

```text
├── ViewModels/        # 앱 상태와 화면 액션 조율
├── Views/

```

## 구현 포인트

### macOS 메뉴바 앱

일반적인 iOS 앱처럼 전체 화면을 중심으로 설계하지 않고, 메뉴바에서 조용히 실행되다가 필요한 순간에 오버레이를 띄우는 흐름으로 구성했습니다.
SwiftUI의 `MenuBarExtra`와 AppKit 윈도우 관리를 함께 사용해 메뉴바 앱과 화면 위 오버레이를 분리했습니다.

### 앱 사용 시간 감지

Screen Time API나 강제 차단 권한을 사용하지 않고, `NSWorkspace.shared.frontmostApplication`과 앱 활성화 알림을 활용해 현재 앞에 떠 있는 앱의 사용 시간을 누적합니다.
MVP에서는 앱 단위 감지를 목표로 하며, 브라우저 내부 탭이나 웹사이트 단위 감지는 제외했습니다.

### 커스텀 캐릭터 애니메이션

사용자가 등록한 이미지 한 장을 억지로 머리/몸통 파츠로 자르지 않고, 이마/턱/목/어깨 기준점을 이용해 움직임을 보정하는 방식으로 접근했습니다.
긴 머리나 캐릭터 장식이 잘리는 문제를 줄이기 위해, 이미지 전체를 유지한 상태에서 mesh deformation을 적용하는 방향으로 구현했습니다.

### 구매 권한 모델

캐릭터 슬롯 비즈니스 모델을 위해 StoreKit 서비스와 앱 내부 권한 모델을 분리했습니다.
개별 슬롯 영구 구매와 전체 슬롯 월 구독을 별도의 권한으로 다루며, UI는 StoreKit 상품 로딩 실패 상태에서도 기본 권한 상태를 유지하도록 설계했습니다.
